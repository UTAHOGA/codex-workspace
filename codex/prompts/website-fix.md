# Website Fix Workflow

Use this workflow for:
- broken links
- Wix embed code
- landing pages
- form buttons
- page layout
- mobile formatting

## Task

Fix the stated website issue.

## Rules

- Preserve existing branding.
- Check all button links.
- Keep code self-contained where possible.
- Do not remove working sections unless asked.

## Final Output

1. What was broken
2. What was changed
3. Final code or files changed
4. Testing checklist
